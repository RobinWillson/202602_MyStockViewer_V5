**\[註1\]:**

byte1,2 代表整數

byte3 代表分子整數部位

byte4 代表分子小數部位

10進制7930393 -\> 16進制00790219

Byte1,2 為0079(16進制) -\> 121(10進制)

Byte3 為02(16進制) -\> 2(10進制)

Byte4為19(16進制) -\> 25(10進制)

以上值組合成字串即為121'02.25
